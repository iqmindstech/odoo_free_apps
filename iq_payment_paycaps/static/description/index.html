Odoo14 - Paycaps Payment Gateway Integration

Feature and Steps

Install payment_paycaps module from APP List

Update Currency code in Currency menu

Ex. UAE: 784 USD: 840

Fill APP ID and Seceret Key in Payment Acquirer

In invoice action click on Generate a Payment link

Copy link and send to customer by email or whatsapp

Fill Card Detail

Invoice Paid

When the payment is success system automatically mark the invoice as paid and reconciles it. No need for anything else !

Customer Payment

Every success payment from Network International Payment Gateway will create a payment in Odoo also along with details.
